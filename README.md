<h1 align="center">📊 Telecom Churn Strategic Analysis</h1>

<p align="center">
  End-to-end <b>Data Analytics</b> & <b>Business Intelligence</b> project focused on
  customer churn analysis in the telecom industry.
</p>

<p align="center">
  This project combines Python-based exploratory data analysis, business-focused storytelling,
  and an interactive dashboard to identify churn drivers, revenue risks, and retention opportunities.
</p>

<hr>

<h2>🚀 Project Highlights</h2>

<ul>
  <li>Performed customer churn analysis using <b>Python</b>, <b>pandas</b>, and data visualization techniques</li>
  <li>Identified high-risk customer segments, including <b>Family Plan</b> and <b>Postpaid Basic</b> users</li>
  <li>Analyzed the relationship between churn and customer behavior such as support calls and payment methods</li>
  <li>Estimated monthly revenue at risk from customer attrition</li>
  <li>Built an interactive HTML dashboard with dynamic Excel upload functionality</li>
  <li>Created recruiter-ready business presentation slides with strategic recommendations</li>
</ul>

<h2>📌 Key Business Insights</h2>

<ul>
  <li>Overall churn rate reached approximately <b>20%</b></li>
  <li>Auto-debit customers showed elevated churn behavior</li>
  <li>Churn probability increased significantly after multiple customer support interactions</li>
  <li>Certain telecom plans demonstrated pricing-to-value mismatch</li>
  <li>Customer experience metrics strongly influenced retention outcomes</li>
</ul>
